整流电路 归纳 总结

整流电路
名称 拓扑电路 条件 公式（注明控制角） 电压、电流波形特点 选取元件耐压、额定电流 注意事项
单相不可控整流
（单相桥式二极管整流）		 4个二极管 $D_1 \sim D_4$ 组成桥式电路；$D_1D_4$ 与 $D_2D_3$ 交替导通 无控制角，二极管自然导通
  • $U_d = 0.9 U_2$
  • $I_d = U_d / R$
  • 每个二极管平均电流 $I_D = I_d / 2$
  • 变压器二次电流有效值 $I_2 = 1.11 I_d$
  • 输出电压 $u_d$：全波脉动直流，无零电压区间
  • 脉动频率 = $2f$（$f$ 为电源频率）
  • 输出电流 $i_d$ 波形与 $u_d$ 同形（电阻负载）
  • 二极管反向重复峰值电压：$U_{RM} \geq \sqrt{2}\, U_2$
  • 额定电流（留1.5~2倍裕量）：$I_F \geq (1.5 \sim 2) \times \dfrac{I_d}{2}$
  • 无需触发电路
  • 输出脉动较大，通常需加滤波
  • 变压器二次侧无直流分量
单相半波
可控整流		 1个晶闸管 $T$ 串联负载；仅利用电源正半周 电阻
负载
  • 控制角 $\alpha$ 范围：$0 \sim \pi$
  • $U_d = 0.45 U_2 \cdot \dfrac{1 + \cos\alpha}{2}$
  • $I_d = U_d / R$
  • 晶闸管导通角 $\theta = \pi - \alpha$
  • $\alpha = 0$ 时 $U_d = 0.45 U_2$
  • $u_d$：仅在 $\alpha \sim \pi$ 区间有正弦输出，其余为零
  • 半波脉动，脉动频率 = $f$
  • $i_d$ 与 $u_d$ 同形
  • $\alpha$ 增大，输出面积减小
  • 晶闸管正向峰值电压：$\sqrt{2}\, U_2$
  • 反向重复峰值电压：$U_{RM} \geq \sqrt{2}\, U_2$
  • 额定电流：$I_{T(AV)} \geq (1.5 \sim 2) I_d$
  • 仅利用正半周，效率低
  • 变压器二次侧有直流分量，铁心易偏磁饱和
  • $\alpha = 0$ 时为半波不可控
阻感
负载
  • 需加续流二极管 $V_{D}$
  • 带续流二极管：$U_d = 0.45 U_2 \cdot \dfrac{1 + \cos\alpha}{2}$
  • 不带续流二极管时，感应电动势维持电流，晶闸管导通角 $\theta > \pi - \alpha$
  • 晶闸管导通期间 $u_d$ 为正弦段
  • 续流期间 $u_d = 0$
  • $i_d$ 因电感平滑作用，波形较平坦
  • $L$ 越大电流纹波越小
  • 晶闸管：$U_{RM} \geq \sqrt{2}\, U_2$
  • 续流二极管：$U_{RM} \geq \sqrt{2}\, U_2$
  • 额定电流留1.5~2倍裕量
  • 必须接续流二极管，防止 $u_d$ 出现负值
  • 无续流管时，电感储能可能损坏晶闸管
  • 变压器二次侧仍有直流分量
单相桥式
全控整流		 4个晶闸管 $T_1 \sim T_4$ 组成全控桥；$T_1T_4$ 正半周导通，$T_2T_3$ 负半周导通 电阻
负载
  • $\alpha$ 范围：$0 \sim \pi$
  • $U_d = 0.9 U_2 \cdot \dfrac{1 + \cos\alpha}{2}$
  • $I_d = U_d / R$
  • 每个晶闸管导通角 $\theta = \pi - \alpha$
  • 晶闸管电流平均值 $I_{T(AV)} = I_d / 2$
  • 变压器二次电流有效值 $I_2 = I_d \sqrt{\dfrac{\pi - \alpha}{2\pi}}$
  • $u_d$：每半周期 $\alpha \sim \pi$ 区间有正弦输出
  • 全波脉动，脉动频率 = $2f$
  • $i_d$ 与 $u_d$ 同形
  • $\alpha$ 增大，输出减小
  • 晶闸管反向重复峰值电压：$U_{RM} \geq \sqrt{2}\, U_2$
  • 额定电流：$I_{T(AV)} \geq (1.5 \sim 2) \times \dfrac{I_d}{2}$
  • 需两组触发脉冲，相差 $180°$
  • 同一桥臂两管不能同时导通（避免短路）
  • 变压器二次侧无直流分量
阻感
负载
  • $L$ 足够大时电流连续
  • $U_d = 0.9\, U_2 \cos\alpha$
  • $\alpha$ 范围：$0 \sim 90°$（整流），$90° \sim 180°$（有源逆变）
  • $I_d = U_d / R$（忽略 $L$ 上压降的稳态值）
  • 每个晶闸管导通角 = $\pi$
  • 晶闸管电流平均值 $I_{T(AV)} = I_d / 2$
  • $u_d$：在 $\alpha$ 较大时出现负电压区间
  • $i_d$：近似恒定直流（$L$ 很大时）
  • 晶闸管电流为矩形波，导通 $\pi$ 弧度
  • 变压器二次电流为正负交替方波
  • 晶闸管：$U_{RM} \geq \sqrt{2}\, U_2$
  • $I_{T(AV)} \geq (1.5 \sim 2) \times \dfrac{I_d}{2}$
  • $\alpha > 90°$ 时 $U_d < 0$，进入有源逆变状态
  • 全控桥不加续流管可实现逆变运行
  • 逆变失败（$\beta < \delta$）会导致短路
单相全波
可控整流
（带中心
抽头变压器）		 变压器中心抽头 + 2个晶闸管 $T_1, T_2$；正半周 $T_1$ 导通，负半周 $T_2$ 导通 电阻
负载
  • $\alpha$ 范围：$0 \sim \pi$
  • $U_d = 0.9 U_2 \cdot \dfrac{1 + \cos\alpha}{2}$
  • 与桥式全控电阻负载公式相同
  • $I_d = U_d / R$
  • 晶闸管平均电流 $I_{T(AV)} = I_d / 2$
  • $u_d$ 波形与桥式电阻负载相同
  • 全波脉动，脉动频率 = $2f$
  • $i_d$ 与 $u_d$ 同形
  • 晶闸管承受反向电压为桥式的 2倍：$U_{RM} \geq 2\sqrt{2}\, U_2$
  • $I_{T(AV)} \geq (1.5 \sim 2) \times \dfrac{I_d}{2}$
  • 晶闸管反向电压是桥式的2倍！
  • 需要带中心抽头的变压器，增加成本
  • 变压器利用率低于桥式
阻感
负载
  • $L$ 足够大，电流连续
  • $U_d = 0.9\, U_2 \cos\alpha$
  • $\alpha$ 范围：$0 \sim 90°$（整流）
  • 每个晶闸管导通角 = $\pi$
  • $u_d$：与桥式阻感负载相同
  • $i_d$：近似恒定直流
  • 晶闸管电流为半波方波
  • $U_{RM} \geq 2\sqrt{2}\, U_2$
  • $I_{T(AV)} \geq (1.5 \sim 2) \times \dfrac{I_d}{2}$
  • 反向电压大是最大缺点
  • 大功率场合一般选用桥式
三相半波
可控整流		 三相变压器二次侧星形连接，3个晶闸管 $T_1, T_2, T_3$ 分别接三相；共阴极接法，公共点接负载 电阻
负载
  • $U_{d0} = 1.17\, U_2$（$\alpha = 0$ 时）
  • $\alpha \leq 30°$（电流连续）：
    $U_d = 1.17\, U_2 \cos\alpha$
  • $\alpha > 30°$（电流断续）：
    $U_d = \dfrac{3\sqrt{2}}{2\pi}\, U_2 \left[1 + \cos\!\left(\alpha + \dfrac{\pi}{6}\right)\right]$
  • 每个晶闸管电流平均值 $I_{T(AV)} = I_d / 3$
  • $\alpha \leq 30°$：$u_d$ 始终大于零，三相脉动连续
  • $\alpha > 30°$：$u_d$ 出现零电压区间
  • 脉动频率 = $3f$
  • 输出脉动比单相小
  • 晶闸管承受最大电压为线电压幅值：$U_{RM} \geq \sqrt{6}\, U_2 \approx 2.45\, U_2$
  • $I_{T(AV)} \geq (1.5 \sim 2) \times \dfrac{I_d}{3}$
  • 自然换相点：滞后相电压过零点 $30°$
  • 变压器二次侧有直流分量，需用曲折形（Zig-Zag）接法消除
  • 输出脉动比单相电路小
阻感
负载
  • $L$ 足够大，电流连续
  • $U_d = 1.17\, U_2 \cos\alpha$
  • $\alpha$ 范围：$0 \sim 90°$
  • 每个晶闸管导通角 = $\dfrac{2\pi}{3}$（120°）
  • $I_{T(AV)} = I_d / 3$
  • $u_d$：$\alpha > 30°$ 时出现负电压区间，但电流不断
  • $i_d$：近似恒定直流
  • 晶闸管电流为 $120°$ 宽矩形波
  • $U_{RM} \geq \sqrt{6}\, U_2$
  • $I_{T(AV)} \geq (1.5 \sim 2) \times \dfrac{I_d}{3}$
  • $\alpha > 30°$ 时 $u_d$ 有负值段
  • $\alpha > 90°$ 时 $U_d < 0$，可进入逆变
  • 变压器利用率随 $\alpha$ 增大而降低
三相桥式
全控整流		 6个晶闸管组成三相全控桥；共阳极组 $T_4, T_6, T_2$（下桥臂）和共阴极组 $T_1, T_3, T_5$（上桥臂）；每次上下各一个管同时导通 电阻
负载
  • $U_{d0} = 2.34\, U_2$（$\alpha = 0$ 时）
  • $\alpha \leq 60°$（电流连续）：
    $U_d = 2.34\, U_2 \cos\alpha$
  • $\alpha > 60°$（电流断续）：
    $U_d = \dfrac{3\sqrt{6}}{2\pi}\, U_2 \left[1 + \cos\!\left(\alpha + \dfrac{\pi}{3}\right)\right]$
  • 每个晶闸管电流平均值 $I_{T(AV)} = I_d / 3$
  • $u_d$：6脉波脉动，脉动频率 = $6f$
  • $\alpha \leq 60°$：$u_d > 0$ 连续
  • $\alpha > 60°$：出现零电压断续
  • 脉动最小，波形质量最好
  • 晶闸管承受电压（线电压幅值）：$U_{RM} \geq \sqrt{6}\, U_2 \approx 2.45\, U_2$
  • $I_{T(AV)} \geq (1.5 \sim 2) \times \dfrac{I_d}{3}$
  • 6个触发脉冲，相邻间隔 $60°$
  • 常用宽脉冲或双窄脉冲触发
  • 自然换相点：线电压交点处
阻感
负载
  • $L$ 足够大，电流连续
  • $U_d = 2.34\, U_2 \cos\alpha$
  • $\alpha$ 范围：$0 \sim 90°$（整流），$90° \sim 180°$（逆变）
  • 每个晶闸管导通角 = $\dfrac{2\pi}{3}$（120°）
  • $I_{T(AV)} = I_d / 3$
  • 变压器二次电流有效值 $I_2 = \sqrt{\dfrac{2}{3}}\, I_d = 0.816\, I_d$
  • $u_d$：6脉波，$\alpha > 60°$ 时出现负电压段
  • $i_d$：近似恒定直流
  • 每相晶闸管电流为 $120°$ 宽矩形波
  • 变压器二次侧线电流为 $120°$ 宽正负交替方波
  • $U_{RM} \geq \sqrt{6}\, U_2$
  • $I_{T(AV)} \geq (1.5 \sim 2) \times \dfrac{I_d}{3}$
  • $\alpha > 90°$ 时 $U_d < 0$，可实现有源逆变
  • 最常用的大功率整流方案
  • 换相重叠角 $\gamma$ 的影响：实际 $U_d = 2.34 U_2 \cos\alpha - \dfrac{3}{\pi} X_B I_d$

DC-DC 斩波电路 归纳 总结

DC-DC 斩波电路（阻感负载）
名称 拓扑电路描述 条件 公式 电压、电流波形特点 选取元件耐压、额定电流 注意事项
降压斩波
（Buck）		 开关管 $S$ 串联在输入与电感 $L$ 之间；续流二极管 $VD$ 并联在 $L$ 和负载之间（阴极接 $L$，阳极接地）；电容 $C$ 并联负载滤波 阻感
负载
  • $D = t_{on} / T$（占空比）
  • $U_o = D \cdot U_i$
  • $0 < U_o < U_i$（降压）
  • 电感电流纹波：$\Delta i_L = \dfrac{(U_i - U_o) D}{f L}$
  • 电感电流连续条件：$L \geq \dfrac{(1-D) R}{2f}$
  • 输出电压纹波：$\dfrac{\Delta U_o}{U_o} = \dfrac{(1-D)}{8 L C f^2}$
  • 电感电流 $i_L$：三角波纹波叠加在直流上
  • $S$ 导通时 $i_L$ 线性上升；$S$ 关断时 $i_L$ 通过 $VD$ 线性下降
  • $u_L$：$S$ 导通时为 $U_i - U_o > 0$；关断时为 $-U_o < 0$
  • 输出电压纹波很小（经 $LC$ 滤波）
  • 开关管：$U_{DS} \geq U_i$，$I \geq I_{o(\max)}$
  • 续流二极管：$U_R \geq U_i$，$I_F \geq I_{o(\max)}$
  • 电感：$L \geq \dfrac{(1-D)R}{2f}$（保证连续）
  • 输出电压极性与输入相同
  • 输出 $\leq$ 输入
  • 效率高，结构简单，应用最广泛
升压斩波
（Boost）		 电感 $L$ 串联在输入端；开关管 $S$ 从电感输出端到地；二极管 $VD$ 串联接到输出端；电容 $C$ 并联负载滤波 阻感
负载
  • $U_o = \dfrac{U_i}{1 - D}$
  • $U_o \geq U_i$（升压）
  • 电感电流纹波：$\Delta i_L = \dfrac{U_i \cdot D}{f L}$
  • 电感电流连续条件：$L \geq \dfrac{D(1-D)^2 R}{2f}$
  • $S$ 导通时：电感储能，$i_L$ 线性上升，二极管截止
  • $S$ 关断时：电感释放能量到负载，$i_L$ 线性下降
  • 输入电流 = 电感电流，连续（纹波较小）
  • 输出电流脉动较大（通过 $VD$ 间断供给负载）
  • 开关管：$U_{DS} \geq U_o$，$I \geq I_L$
  • 二极管：$U_R \geq U_o$，$I_F \geq I_o$
  • 电感：$L \geq \dfrac{D(1-D)^2 R}{2f}$
  • 输出极性与输入相同
  • $D$ 不能取1（否则短路）
  • 实际升压比受限于寄生参数
  • 输入电流连续，适合光伏、电池升压
升降压斩波
（Buck-Boost）		 开关管 $S$ 接在输入端与地之间；电感 $L$ 一端接 $S$ 与 $VD$ 的公共点，另一端接输出；二极管 $VD$ 反向并联；输出极性与输入相反 阻感
负载
  • $U_o = \dfrac{D}{1-D} \cdot U_i$
  • $D < 0.5$：降压；$D = 0.5$：$U_o = U_i$；$D > 0.5$：升压
  • 电感电流纹波：$\Delta i_L = \dfrac{U_i \cdot D}{f L}$
  • 电感电流连续条件：$L \geq \dfrac{(1-D)^2 R}{2f}$
  • $S$ 导通时：输入给 $L$ 充电，$i_L$ 上升，$VD$ 截止
  • $S$ 关断时：$L$ 通过 $VD$ 向负载放电，$i_L$ 下降
  • 输入电流和输出电流均为脉动的
  • 开关管：$U_{DS} \geq U_i + U_o$
  • 二极管：$U_R \geq U_i + U_o$
  • $I \geq I_{L(\max)}$
  • 输出电压极性与输入相反！
  • 可升可降，应用灵活
  • 输入输出电流均为脉动，EMI较大
Cuk斩波 输入端串联电感 $L_1$；开关管 $S$ 接地；耦合电容 $C_1$ 连接两级；二极管 $VD$；输出电感 $L_2$ 串联负载；输出电容 $C_2$ 滤波 阻感
负载
  • $U_o = \dfrac{D}{1-D} \cdot U_i$
  • 与 Buck-Boost 电压增益相同
  • $C_1$ 上电压：$U_{C1} = U_i + U_o = \dfrac{U_i}{1-D}$
  • $L_1$ 电流纹波：$\Delta i_{L1} = \dfrac{U_i D}{f L_1}$
  • $L_2$ 电流纹波：$\Delta i_{L2} = \dfrac{U_o (1-D)}{f L_2}$
  • $S$ 导通：$L_1$ 由输入充电；$C_1$ 向 $L_2$ 和负载放电
  • $S$ 关断：$L_1$ 和输入向 $C_1$ 充电；$L_2$ 向负载放电
  • 输入电流（$i_{L1}$）连续
  • 输出电流（$i_{L2}$）连续
  • $S$：$U_{DS} \geq U_i + U_o$
  • $VD$：$U_R \geq U_i + U_o$
  • $C_1$ 耐压 $\geq U_i + U_o$
  • $I_S = I_{L1} + I_{L2}$
  • 输出极性反转（与输入相反）
  • 输入输出电流均连续，优于 Buck-Boost
  • 通过 $C_1$ 传递能量
  • $L_1$、$L_2$ 可耦合以降低纹波
Sepic斩波
（Single-Ended Primary-Inductor Converter）		 输入串联电感 $L_1$；开关管 $S$ 接地；耦合电容 $C_1$ 连接 $L_1$ 输出端与 $L_2$；$L_2$ 经二极管 $VD$ 接输出；输出电容 $C_2$ 滤波 阻感
负载
  • $U_o = \dfrac{D}{1-D} \cdot U_i$
  • 与 Buck-Boost、Cuk 电压增益相同
  • $C_1$ 上电压：$U_{C1} = U_i$
  • $S$ 导通：$L_1$ 储能；$C_1$ 通过 $L_2$ 放电
  • $S$ 关断：$L_1$ 给 $C_1$ 充电并向负载供电；$L_2$ 也向负载供电
  • 输入电流连续（经 $L_1$）
  • 输出电流脉动（经 $VD$ 间断导通）
  • $S$：$U_{DS} \geq U_i + U_o$
  • $VD$：$U_R \geq U_i + U_o$
  • $C_1$ 耐压 $\geq U_i$
  • 输出极性与输入相同（不反转）
  • 可升可降
  • 输入电流连续，适合电池、光伏
  • 效率略低于 Cuk（输出电流脉动）
Zeta斩波
（Inverse SEPIC）		 开关管 $S$ 接在输入端；耦合电容 $C_1$ 连接 $S$ 输出端与 $L_2$；二极管 $VD$ 从 $C_1$与$L_2$ 连接点到地；$L_2$ 串联输出；电容 $C_2$ 滤波 阻感
负载
  • $U_o = \dfrac{D}{1-D} \cdot U_i$
  • 与 SEPIC 互为对偶电路
  • $C_1$ 上电压：$U_{C1} = U_i$
  • $S$ 导通：输入通过 $C_1$ 给 $L_2$ 充电并供给负载
  • $S$ 关断：$L_2$ 通过 $VD$ 续流向负载供电
  • 输入电流脉动（开关直接串联输入）
  • 输出电流连续（经 $L_2$ 滤波）
  • $S$：$U_{DS} \geq U_i + U_o$
  • $VD$：$U_R \geq U_i + U_o$
  • $C_1$ 耐压 $\geq U_i$
  • 输出极性与输入相同（不反转）
  • 可升可降
  • 输出电流连续，输入电流脉动
  • 与 SEPIC 互为对偶
